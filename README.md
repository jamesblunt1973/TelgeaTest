## Backend: NodeJS with Express

### Make sure you have NodeJs version 24 or higher
### Install dependencies by running: `npm i`
### `npm run lint-format` for linting & formatting provided by [Biome](https://biomejs.dev/)
### Run test by `npm test`
### Run the server by executing command: `npm start`
### Use postman to call the [api endpoint](http://localhost:3000/normalized)

## Frontend: Angular 20 (zoneless)

### Run `npm test`
### Run `npm start` or `ng serve`
